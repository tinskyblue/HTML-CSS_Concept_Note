# 글자와 관련된 태그

## `<strong>`볼드체, `<em>`이탤릭체
```
<strong>볼드체</strong><br>
<em>이탤릭체</em>
```
<pre>
<strong>볼드체</strong><br>
<em>이탤릭체</em>
</pre>

```
<p>일반<sup>위치보다 위로</sup></p>
<p>일반<sub>위치보다 아래로</sub></p>

```
<pre>
<p>일반<sup>위치보다 위로</sup></p>

</pre>
